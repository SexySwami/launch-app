# Launch — Mission Control for Momentum

A gamified task-initiation web app. Type one task, and the app breaks it into 4
guided checkpoints with a 3-2-1-GO countdown and a momentum score.

Mobile-first dark UI. Built as a single self-contained HTML file using
React 18 + Babel via CDN — no build step.

## Run locally

```bash
open index.html
```

Or serve from any static server.

## Deploy

Deployed via Vercel as a static site (no framework, no build).
